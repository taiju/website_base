h1. README

h1. これは何？

Webサイト制作の基本セットを管理しているリポジトリ。

h2. 利用技術

* git
* sass
* yuicompressor
* Mojolicious::Lite

h2. 使い方

# このリポジトリをクローンする。
# <code>templates/layouts/</code> で基本レイアウトを作る。
# <code>templates/*.html.ep</code> でHTMLテンプレートを作る。
# <code>app</code>（Mojolicious lite_app） でルーティングする。
# <code>_scss/</code> にあるscssファイルでスタイルを書く。
# <code>_scss/style.scss</code> を sassを使って <code>public/style.css</code> に出力する。
# スタイルはページごとに <code>_scss/_pagename.scss</code> に分け、 <code>_scss/style.scss</code> でインポートする。
# <code>_expand_js/</code> にあるJSファイルでスクリプトを書く。
# <code>_expand_js/site.js</code> を yuicompressorを使って <code>public/js/site.min.js</code> に出力する。
